Technical Methodology & Comparison Landscape

1. Sourcing Data & Accuracy

Nutri Nexus is built on verified research-grade datasets rather than generic Western calorie counters. It programmatically synthesizes the following core clinical sources:

  • ICMR-NIN IFCT 2017: 2,887 raw agricultural and primary ingredients, accounting for Indian agronomic profiles (soil composition, local crops).
  • INDB (Indian Nutrient Databank) 2024: 1,014 cooked Indian recipes calculated via nutrient retention factors.
  • ICMR-NIN 2024 Guidelines: Recommended Dietary Allowances (RDA) and Asian Indian BMI standards.

3. Key Differentiators of Nutri Nexus

Dual Database Synthesis: Bridges the gap between raw programmatic libraries (like ifct2017) and cooked-recipe datasets (like INDB 2024), mapping them into a unified search index.
Clinical Target Customization: Rather than generic 2,000 kcal targets, Nutri Nexus calculates physical-activity and physiological-status RDA requirements (pregnancy T1/T2/T3, lactation) and uses the official Asian Indian BMI limits.
Linguistic & Portion-Size Resolution: Supports searches in 17 regional Indian languages and translates qualitative servings (Piece, Bowl, Katori, Cup) into gram weights via energy-density formulas.

4. Scientific Formulas Used

Serving-Size Pre-computation (Offline, Data-Generation Step):

The formula below was applied once offline during database construction to convert IFCT energy-density ratios into absolute gram weights stored as the servingSize field in foodData.js. It does not run at query time.

servingSize_g = round( (unit_serving_energy_kcal / energy_per_100g_kcal) × 100 )

Runtime Portion Calculation (Live, at Query Time):

When a user enters a quantity with a qualitative unit (e.g. “2 Katori”), the app performs a simple lookup multiplication using the pre-stored value:

weight_g = quantity × food.servingSize
nutrient = (weight_g / 100) × food.nutrient_per_100g

RDA Target Dynamic Computation:

Nutrient targets are computed live from user profile inputs (age, gender, physical activity, body weight). Pregnancy increments follow ICMR-NIN 2020 Table 3 trimester-specific values: +150 kcal (T1), +250 kcal / +8g protein (T2), +350 kcal / +15g protein (T3). Elderly users (≥60 yrs) receive calcium raised to 1,200 mg and protein scaled at 1.0 g/kg to counteract sarcopenia and bone loss.
